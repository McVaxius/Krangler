## What is Krangler?

Krangler is a Dalamud plugin for FFXIV that randomizes visible player information to make your screenshots and streams safe for sharing. No more worrying about accidentally exposing someone's character name!

## Features

- **Krangle Names** — Replaces all visible player names with randomized exercise-themed words (same player always gets the same fake name per session)
- **Krangle Genders** — Randomizes displayed gender of visible players *(requires Glamourer)*
- **Krangle Races** — Randomizes races and subraces for visible players *(requires Glamourer)*
- **Krangle Appearance** — Randomizes hair, face, eyes, etc. *(requires Glamourer)*
- **Super Krangle Master 4000** — Transform everyone into special NPC models! *(requires Glamourer)*
- **DTR Bar** — Quick toggle from the server info bar
- **Simple UI** — One window, clear checkboxes, done

## Commands

| Command | Description |
|---------|-------------|
| `/krangler` | Open the Krangler window |
| `/kr` | Open the Krangler window |
| `/kr on` | Enable krangling |
| `/kr off` | Disable krangling |
